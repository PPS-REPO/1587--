# 문제

Geometry Dash에서 NLW(Non-Listworthy) 맵은 출시되고 단 한 번도 $m$위 안에 든 적 없는 맵이다.
예를 들어 $m=3$이고 맵의 난이도가 (출시된 순서대로) 100, 200, 300, 89, 350, 120, 400, 299이면 NLW맵은 난이도가 89, 120, 299인 맵이다.
자세히 설명하자면

1. 난이도가 100인 맵이 출시된다. 현재 맵 리스트는 (난이도 순서대로) [100]이다.
2. 난이도가 200, 300인 맵이 출시된다. 맵 리스트는 [300, 200, 100]이다. 이 3개의 맵은 3위 안에 한 번이라도 든 적이 있기 때문에 NLW 맵이 아니다.
3. 난이도가 89인 맵이 출시된다. 맵 리스트는 [300, 200, 100, 89]이다.
...
4. 난이도가 299인 맵이 출시된다. 맵 리스트는 [400, 350, 300, 299, 200, 120, 100, 89]이다.

여기까지 맵 리스트를 채우는 동안 한 번도 3위 안에 든 적이 없는 맵(NLW 맵)은 난이도가 89, 120, 299인 맵이다. 따라서 3을 출력하면 된다.

# 입력

첫째 줄에 $n$과 $m$이 입력된다. 둘째 줄에 $n$개의 Geometry Dash 맵의 난이도가 출시된 시간 순서대로 입력된다. 맵의 난이도는 $10^9$ 이하의 자연수이다. 어느 두 맵도 난이도가 같지 않다.

* $1 \le m \le n$
* $1 \le n \le 300000$

# 출력

첫째 줄에 NLW 맵의 개수를 출력한다.
